第二章笔记

发表于 2021-03-11 | 更新于: 2021-03-11 | 分类于 c primer plus

程序的分析

  1. #include <stdio.h> :包含另一个文件，也成为预处理命令
  2. int main(void) :函数名
  3. {} :花括号中为函数体
  4. int num:声名
  5. printf :调用函数
  6. 其中的// 和 /* */分别为单行注释和多行注释
需要注意的一点

在旧的规则中要求将所有的变量声名放在顶部但是c99可以支持将声明放在程序的任何位置

关于标识符

  1. 在c语言中标识符由字母，数字，和下划线组成而且不能以数字开头

提高程序可读性的技巧

  1. 选择有意义的函数名和谐注释
  2. 在函数中用空行分隔概念上的多个部分
  3. 每条语句占一行
编译器通常可以警告我们语法错误，而语义的错错误则需要我们自己调试发现
